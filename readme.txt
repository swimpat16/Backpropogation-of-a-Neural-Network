Description:
	Simulator for backpropogation in a neural network written in C++ be Alex Chaloux and Patrick Elam.
File:
	Data.xlsm: Contained a script that would pull the essential data from all of our output files and drop it into one sheet. The script then created graphs that allowed for visualization of the data.
	gen_config1.cc: Created config files for problem 1
	gen_config2.cc: Created config files for problem 2
	lab4.cc: Simulator. Takes the output file name as a parameter. Other parameters are taken as inputs as it runs.
	run.cc: Script used to run all configurations through the simulator.
	testing, training, and validation : Inputs and correct outputs used to test, train, and validate the simulator
Usage: lab4 outputfilename
